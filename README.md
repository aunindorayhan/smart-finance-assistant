# Smart Finance Assistant

## Project Overview

This project is a Smart Finance Assistant developed in Python using Google Colab. The assistant cleans transaction data, handles messy currency formatting, separates refunds from normal spending, calculates category summaries, compares spending against a weekly budget, and generates simple financial insights.

The project demonstrates how AI-assisted development can support a business information systems task. It follows a structured development process: understanding the problem, identifying inputs and outputs, working through manual examples, writing pseudocode, converting the solution into Python, testing, debugging, and reflecting on the final result.

## Key Features

- Cleans transaction data
- Handles `$` and `AUD` currency formatting
- Separates refunds from spending
- Calculates total positive spending, refunds, and net spending
- Summarises spending by category
- Identifies the highest spending category
- Compares spending against a weekly budget
- Generates plain-English financial insights
- Includes testing and debugging evidence
- Documents AI collaboration and reflection

## Technologies Used

- Python
- Google Colab
- pandas
- GitHub
- AI-assisted development tools

## How to Run the Project

1. Open the notebook file in Google Colab.
2. Run the setup and import cells.
3. Run the sample transaction data cell.
4. Run the spending analysis functions.
5. Run the recommendation and assistant-style functions.
6. Run the testing cells.
7. Review the output, testing summary, and reflection sections.

## AI Collaboration

AI was used as a development partner throughout this project. It helped with brainstorming, pseudocode, Python implementation, debugging, testing ideas, and documentation.

I did not simply accept AI responses without checking them. I reviewed whether the outputs matched the finance problem, tested the code with sample data, fixed errors, and adapted the solution to make it more reliable and business-focused.

## Testing and Debugging

The assistant was tested with normal transaction data, refunds, messy currency values, invalid data, missing required columns, and weekly budget scenarios.

The testing confirmed that the assistant can clean data, separate refunds, calculate spending summaries, compare spending with a budget, and produce clear financial insights.

## Final Reflection

This project shows how AI can support practical software development when used critically. The final Smart Finance Assistant demonstrates data processing, financial analysis, user-centred design, testing, debugging, documentation, and AI-assisted development skills.
