# AI Evidence Notes

This folder organises evidence of AI-assisted development for the Smart Finance Assistant project.

AI was used to support:
- brainstorming the finance assistant problem and user needs
- planning inputs, outputs, and manual examples
- improving Python functions for transaction cleaning and spending analysis
- debugging notebook execution issues
- creating testing ideas for normal, edge, and error cases
- refining the README and Developer's Diary

The final notebook and Developer's Diary explain how AI suggestions were reviewed, tested, and adapted rather than accepted without checking.
